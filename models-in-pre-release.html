TushAI T1 Search in Beta and T1.1 in Alpha

Published on: April 5, 2025

TushAI is proud to announce the pre-release of our next model in both beta and alpha phase. The TushAI T1 search and TushAI T1.1 has been released for pro users.

TushAI T1.1

We are developing a successor to the T1 series by making this new series. Currently development stage is still in Alpha. The TushAI T1.1 according to benchmarks done by TushAI internal labs is showing massive performance improvements in AI benchmarks beating competitors. Currently the TushAI T1.1 is still on the alpha stage because of the model which is not fully refined which at this current stage, is showing flaws in speed and has some rate limits.

TushAI T1 Search

One of the limitations listed in the TushAI T1 release page is the limited knowledge in pop culture. We realised one of the ways to overcome this limitations is to implement a search feature to the TushAI T1 model which can provide the AI model with current information and assist the AI model to get the information and knowledge about a certain question. This model has been tested and the results shows that this new model has a wider range of knowledge about pop culture and things that the current model has limitations on. This AI model is only currently in Beta because the search feature of this AI is slightly resource intensive. We are working to use the compute resource more efficiently when using the search feature before releasing this model for free users.

Estimation of Release

We don't have a exact release date of these AI models but estimates for Tush T1 Search is before July and for the TushAI T1.1 is this year but we do not have a more precise estimate.
